# Multi-Feature Mobile Application (.NET MAUI)

This project is a multi-page cross-platform mobile application developed using .NET MAUI.  
It demonstrates modern mobile development practices such as API integration, local data management, and responsive user interface design.

---

## Features

- News Module  
  - Fetches and displays news via RSS feeds  
  - Detailed news view using WebView  

- Weather Module  
  - City-based weather tracking  
  - Dynamic city list with local persistence (JSON)  

- Exchange Rates  
  - Displays up-to-date currency data  

- To-Do List  
  - Add, update, delete, and complete tasks  
  - Local data management  

- Authentication  
  - Login and Register pages (UI and basic flow)  

- Settings Page  
  - Application-level configurations  

---

## Technologies Used

- .NET MAUI  
- C#  
- XAML  
- REST APIs / RSS Feeds  
- JSON-based local storage  

---

## Project Structure

- Models/ → Data models  
- Services/ → API and business logic  
- Views/ → UI pages  
- Resources/ → Assets (images, fonts, styles)  
- Platforms/ → Platform-specific configurations
